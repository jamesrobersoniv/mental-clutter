# Elephant.art CSS Files

## Downloaded Files

- `main.css`
- `elephant.min.css`
- `flex-masonry.css`
- `inline-styles.css`

## Usage in Hugo

Add to `layouts/_default/baseof.html`:

```html
<link rel="stylesheet" href="{{ "css/elephant-art/main.css" | absURL }}">
<link rel="stylesheet" href="{{ "css/elephant-art/elephant.min.css" | absURL }}">
<link rel="stylesheet" href="{{ "css/elephant-art/flex-masonry.css" | absURL }}">
```
